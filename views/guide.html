Eto ne API E123x guide

Routes

Eto ne API E123x has these routes:

  • /get-folder-data/* — returns info about directory structure in content folder;

  • /happy-norming/ — returns cute picture with a wish of a good day;

  • /funny-animals/ — returns funny picture with animals;

  • /parser/ — return parsed data from html-pages;
